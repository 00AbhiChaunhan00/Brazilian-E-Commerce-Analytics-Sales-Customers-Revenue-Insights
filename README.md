# 📊 Business Performance Analysis of Brazilian E-Commerce using SQL & Python

## 📌 Project Overview

This project analyzes a real-world Brazilian E-Commerce dataset (Olist) to uncover valuable business insights related to customer behavior, sales performance, payment methods, product categories, sellers, and revenue trends.

The analysis was performed using **MySQL** for querying and **Python** for data analysis and visualization. The objective is to demonstrate how SQL and Python can be combined to solve real business problems and support data-driven decision making.

---

# 📂 Dataset

**Dataset Source**

Brazilian E-Commerce Public Dataset by Olist

The dataset contains information about:

- Customers
- Orders
- Order Items
- Payments
- Products
- Sellers
- Product Categories
- Geolocation

Total Records:
- 100,000+ Orders
- 90,000+ Customers
- Multiple relational tables

# 📌 Key Business Insights

Some major insights obtained from the analysis include:

- Credit Cards are the most preferred payment method.
- A small number of sellers contribute a large share of total revenue.
- Low-priced products are purchased significantly more frequently.
- Certain states contribute disproportionately higher revenue.
- Customer purchasing behavior varies considerably across cities.
- Monthly sales exhibit clear seasonal trends.
- A small percentage of customers account for a large portion of total revenue.
